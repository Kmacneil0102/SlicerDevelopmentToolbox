SlicerDevelopmentToolboxUtils
=============================

The SlicerDevelopmentToolbox is an Python API that simplifies the development of Slicer extensions. It consists of the
following components:

* Mixins
* Helpers
* Widgets (QT)
* Buttons (QT)
* Icons (QT)
* Constants (DICOM, QT, CSS)
* Decorators
* Events (VTK)
* Exceptions
* Metaclasses
